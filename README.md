# vr-scene-threejs
3D rendering with animation, camera setup, lighting, and textures in threejs

## Keyboard Controls

### Avatar controls
`w` : Start moving forward

`s` : Stop moving

`a` : Turn Left

`d` : Turn Right

`space` : Jump

`f` : Attach to train

`click` : Change avatar's appearance

### Camera controls
`c` : Toggle Camera [Fixed / Avatar POV]

`Mouse Drag` : Camera Orbital Controls
